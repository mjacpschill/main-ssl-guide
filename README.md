# main-ssl-guide

本仓库用于归档和发布多个独立的 HTML 页面，内容主要围绕 SSL/TLS 相关主题。

## 项目简介

这是一个以静态 HTML 页面为主的文档归档仓库。仓库中的每个页面均为独立文件，不依赖于特定域名或具体网站，仅作为通用技术资料的整理与展示。

## 目录结构

```
main-ssl-guide/
├── index.html          # 仓库首页/导航
├── pages/              # 存放各独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 共用资源文件（CSS、图片、字体等）
└── README.md           # 本文件
```

- `index.html`：仓库的入口页面，提供页面列表或导航链接。
- `pages/`：所有独立 HTML 页面均放置在此目录下，每个页面可自包含样式与脚本。
- `assets/`：存放所有页面可能共用的静态资源，如图片、样式表或 JavaScript 文件。

## 页面归档说明

- 每个 HTML 页面均为独立完整的文档，可直接通过浏览器打开查看。
- 页面内容聚焦于 SSL/TLS 相关技术说明、配置示例、常见问题或参考笔记。
- 归档不针对任何特定域名、品牌或服务，仅作为通用技术资料。
- 页面命名尽量保持清晰、简洁，便于索引与引用。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时请保持目录结构一致。
- 如需提交贡献，请确保页面内容不包含具体域名、营销信息或诱导点击内容。
- 建议在新增页面时，同时更新 `index.html` 中的导航列表。
- 所有页面应保持基本的可读性与兼容性，避免依赖外部在线资源（除非必要）。

## 使用说明

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 或 `pages/` 下的任意 HTML 文件即可浏览。
3. 也可通过 GitHub Pages 或其他静态托管服务发布本仓库内容。
